= `mean`

`mean` is a function that calculates the average of a vector of values.

== Examples

=== How do I get the average of a vector of values?

.Click to see solution
[%collapsible]
====
[source, R]
----
mean(c(1,2,3,4))
----
[source, R]
----
[1] 2.5
----
====

=== How do I get the average of the values in a vector when some of the values are: `NA`, `NaN`?

.Click to see solution
[%collapsible]
====
Many R functions have the `na.rm` argument available. 
This argument is "a logical value indicating whether NA values should be stripped before the computation proceeds. (https://stackoverflow.com/questions/58443566/what-does-na-rm-true-actually-means[Citation])

[source, R]
----
mean(c(1,2,3,NaN), na.rm=T)
----
[source, R]
----
[1] 2
----
[source, R]
----
mean(c(1,2,3,NA), na.rm=T)
----
[source, R]
----
[1] 2
----
[source, R]
----
mean(c(1,2,NA,NaN,4), na.rm=T)
----
[source, R]
----
[1] 2.333333
----
====
