= TDM 40200: Project 13 - Working with Ollama

== Project Objectives
This project introduces you to working with local large language models (LLMs) using Ollama. You will learn how to run models, compare their outputs, evaluate their limitations, and customize their behavior through prompting.

.Learning Objectives
****
- Run and interact with local LLMs using Ollama  
- Download and compare multiple models (e.g., `llama3.2` and `mistral`)  
- Measure and evaluate model performance and response quality  
- Identify hallucinations in model outputs and explain their implications  
- Use external data sources (e.g., PDFs, vector databases) to enhance model responses  
- Modify system prompts to control tone, style, and personality of model outputs  
****
