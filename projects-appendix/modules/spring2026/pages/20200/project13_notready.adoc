=== Question 2 - Introduction to Geospatial Data & GeoPandas (2 points)

In our first question, we identified and cleaned coordinate data so now have valid data to work with. If, for instance, we wanted to know where clusters occur for crashes, we would need geospatial tools. Clustering here refers to a technique that groups data points using similar geographical features and proximity. 

Working with geospatial data in Python can be done by using the https://geopandas.org/en/stable/getting_started/introduction.html[GeoPandas] library. We can think of GeoPandas as a combination of the Pandas library and geometry. It has the abilites to perform spatial operations, geometric manipulations, and create maps. 

[big]#*Geometry Types*#

There are three main geomtry types for Geospatial data

1. Points: Single (x,y) pair identifying a specific location. 
- Example: A crash that happened at a latitude/longitude.
2. Lines: Ordered set of connected coordinate pairs. Lines have length unlike points, but do not have an enclosed area like polygons do. 
- Example: A road.
3. Polygons: Series of points forming a closed loop. 
- Example: A state.

These are also the three basic geometric object classes in GeoPandas. 

[big]#*https://geopandas.org/en/stable/docs/reference/api/geopandas.GeoDataFrame.html[GeoDataFrame]*#

GeoDataFrame is an extension of the Pandas dataframe to accomodate geospatial vector data. It works the same way except there is an additional 'geometry' column storing shapes that represent Earth's physical locations. 

We already have GeoPandas on Anvil. Just import with:

[source,python]
----
import geopandas as gpd
----

[source,python]
----
new_df = gpd.GeoDataFrame(df_clean, geometry=gpd.points_from_xy(df_clean['LONGITUD'], df_clean['LATITUDE']), crs='EPSG:4326')
----

- `gpd.GeoDataFrame()`: returns the dataframe containing geometry. We passed in EPSG:4326 as 'crs' (Coordinate Reference System). EPSG:4326 is the "World Geodetic System 1984", which is the standard and the most widely used coordinate reference system. 

- `geometry=gpd.points_from_xy(df_clean['LONGITUD'], df_clean['LATITUDE'])`: https://geopandas.org/en/stable/docs/reference/api/geopandas.points_from_xy.html[`.points_from_xy()`] takes in the longitude and latitude columns and converts into Point objects. It creates a GeometryArray directly.

https://shapely.readthedocs.io/en/stable/[Shapely] is a Python library meant to perform spatial operations on Cartesian plane (2D surface). It does not work with coordinate reference systems so for those aspects, it gets combined with GeoPandas.
