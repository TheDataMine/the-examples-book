
= 4. Joins: Combining Tables


== Concepts

Primary key / foreign key. 

INNER JOIN, LEFT JOIN. 

Why/when we join tables. 

The `JOIN` command lets us combine data from two or more tables by matching values in a shared column. 

== Demo

Let us simply join two tables firs:

[source, sql]
----
SELECT * FROM crew as s INNER JOIN people as p
ON s.person_id = p.person_id LIMIT 5;
----
